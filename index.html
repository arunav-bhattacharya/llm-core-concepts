Phase 0 · Python Bridge

Week 0–1 · ~15h. You're a senior JVM dev — you need idiomatic Python, not CS fundamentals. Working fluency is the exit bar.

Topics & depth

async / await + asynciodeep — every agent framework is async-first
Type hints + Pydanticdeep — structured outputs run on it
uv + virtualenvs + Jupytersetup fluency
Decorators, context managers, httpxread/write comfort
pandas / numpylight — skim only
AnalogyPydantic ≈ Kotlin data classes + Bean Validation in one. async def ≈ suspend functions with an explicit event loop. Decorators ≈ annotations that actually execute. Map, don't relearn.
GotchaDon't spend 3 weeks "finishing Python first." Ed Donner's courses teach tooling setup in-line; start Phase 1 the moment you can read async code.
