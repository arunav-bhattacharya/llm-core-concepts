## How it's published

- [`index.html`](index.html) is the complete, self-contained page (inline CSS/JS).
- [`.github/workflows/deploy-pages.yml`](.github/workflows/deploy-pages.yml) builds and
  deploys it to GitHub Pages on every push to `main`.
